Discuss the scope of var, let, and const

"var" keyword can be reassigned with the same global scope of memory block. More than value cannot be declared similar variable named block scope in "let" keyword. "const" is also a block scope and it only can assign one value

Tell us the use cases of null and undefined

null: it occurs when the variable is declared as "null" value. Therefore, later the null value can be replaced in the database if any data needs to be inserted. undefined: if the object did not have corresponding value, then it will return "undefined" result. Same case can happened with array and variable assigning time.

What do you mean by REST API?

The REST API allocate an URL with data resources where these data can be manipulated with certain allocated operations with CRUD system. This CRUD system represent Create, Read, Update and Delete which is similar as POST, GET, PUT and DELETE.
